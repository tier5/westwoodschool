== Description ==

Analytics of Google: analytics code integration on WordPress website

= Google Analytics features =

* Automatic Google Analytics code integration<br />
  Past your Google Analytics Tracking ID and the Google Analytics Code will be generated and integrated automatically.<br />
* Manual Google Analytics code integration<br />
  Past your Google Analytics Code and Google Analytics plugin will integrate it in WordPress website automatically.<br />

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory;
2. Activate the plugin through the 'Plugins' menu in WordPress;
3. In your WordPress go to "Settings" -> "Google Analytics Code" and enter your Google Analytics Code there.
